# Abschlussprojekt SQL

SQL-Abschlussprojekt mit Datenaufbereitung, SQL-Skripten und Ergebnispräsentation.

## Projektziel
Den bereitgestellten Kreditrisiko-Datensatz für SQL-Analysen aufbereiten und zentrale Fragestellungen mit strukturierten Abfragen auswerten.

## Dateien
- `SQL Abschluss.sql` – SQL-Skript mit den zentralen Abfragen
- `credit_risk_dataset_org.csv` – Originaldaten
- `credit_risk_dataset_sql_ready_v2.csv` – aufbereitete Version für SQL
- `Abschlussprojekt Modul Datenbank und SQL.docx` – Projektdokumentation
- `ProjektA_Handout_Styled.pdf` – Handout
- `Abschluss 1.pptx` – Präsentation

## Nutzung
1. CSV-Datei in dein SQL-System importieren
2. `SQL Abschluss.sql` ausführen
3. Ergebnisse mit Dokumentation und Präsentation abgleichen

## Tech Stack
- SQL
- CSV-Datenaufbereitung
- Dokumentation/Präsentation
